# Vision

Design Language (DSL) is an open specification for human and AI-assisted communication.

Its goal is to make communication more consistent, intentional, and human across writing, interfaces, documentation, design artifacts, and AI-assisted workflows.

## Two horizons

### DSL 1.x

Deliver a practical, stable specification people can use now for audits, revisions, and cross-tool consistency.

### DSL 2.0

Evolve DSL into an AI-native communication platform with typed rules, schemas, inheritance, project overrides, generated adapters, validation, linting, a CLI, reusable language components, and multiple implementations.

The 2.0 vision should guide architecture without slowing useful 1.x releases.

## North star

The quality of communication should not depend on which AI generated it.
