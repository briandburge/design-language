# Project context

Design Language is a portable editorial and interface language system for human and AI-assisted workflows.

Its immediate purpose is to help people:

- make language more consistent
- reduce generic AI writing patterns
- audit documents and websites without rewriting them
- apply the same language rules across AI-assisted workflows

The current public release is DSL 1.0.0.

The current priority is to make the 1.x specification useful and reliable before beginning DSL 2.0.

The long-term direction is an AI-native communication specification with structured rules, generated adapters, validation, linting, and project inheritance.
