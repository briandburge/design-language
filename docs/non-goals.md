# Non-goals

DSL is not:

- a replacement for editorial judgment
- an AI detector
- a grammar checker
- a marketing framework
- a prompt library
- tied to one model, vendor, tool, or programming language
- intended to remove voice, style, or personality

DSL does not treat every flagged term as a violation.

DSL does not optimize for sounding human. It optimizes for communication that is intentional, specific, consistent, and appropriate to context.
