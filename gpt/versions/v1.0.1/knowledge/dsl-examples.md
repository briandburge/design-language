# Transformations

## Promotional to factual

Before: `We created a cutting-edge platform that transformed onboarding.`

After: `The new flow reduced onboarding time by 37%.`

Rules: DL-041, DL-042, DL-065

## False contrast to direct statement

Before: `It is not about adding features. It is about creating value.`

After: `We prioritized the features that reduced setup time.`

Rules: DL-061, DL-066

## Weak error to actionable error

Before: `Invalid request.`

After: `We could not submit the form. Review the highlighted fields and try again.`

Rules: DL-101, DL-102
