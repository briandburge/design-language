# Feature release instructions for Codex

## Objective

Accomplish both goals:

1. Keep DSL 1.x useful, testable, and easy to refine.
2. Preserve the long-term vision for a mature AI-native DSL platform.

## Required reading

Read `AGENTS.md`, `README.md`, all files in `docs/`, `rules/rules.yaml`, `audit/`, and `decisions/`.

## Before editing

Summarize:

- DSL 1.x purpose
- DSL 2.0 vision
- Sprint 1 sequence
- proposed branch
- expected file changes
- Custom GPT impact

## After editing

Provide:

- validation results
- changed files
- diff summary
- assumptions
- release impact
- Custom GPT files to replace
- GPT instruction changes
- conversation-starter changes
- known limitations

Do not commit, push, merge, tag, or publish without approval.
